🧮 Calculator Web Application

A simple, responsive, and interactive calculator developed using HTML, CSS, and JavaScript. This web application allows users to perform basic mathematical calculations through an easy-to-use and modern interface.

📌 Features

Perform basic arithmetic operations (Addition, Subtraction, Multiplication, and Division)

Percentage (%) calculation

Clear (C) button to reset calculations

Delete (⌫) button to remove the last entered value

Responsive design for different screen sizes

Real-time display of input and results


🛠️ Technologies Used

HTML

CSS

JavaScript


🚀 How to Run the Project

1. Download or clone the repository.


2. Open the project folder.


3. Run the index.html file in any web browser.
